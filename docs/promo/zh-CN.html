claude-stray — 给你的 Claude Code 工作做个仪表盘

写给"我有 200 个 session jsonl,记不起来上周在搞什么"的人

我有 832 个 ~/.claude/projects/*.jsonl。其中可能 30 个是 "我在认真做的事",剩下都是 9 个月前问过一次然后忘了的。 claude --resume 弹出来的列表按时间倒序排,纯 UUID,翻三屏 都看不到我上周追到一半的那个 HSF 卡点排到哪儿了。

做了个东西叫 claude-stray。它把所有 session 用 Haiku 4.5 自动聚类成 "initiative"(可以理解成"小项目"),每个 initiative 一张卡片,卡片上有摘要、进度、任务清单、卡点、相关的 CR/MR 链接、 能直接点 🆕 在新 pane 里 resume 的 session 列表。打开 http://127.0.0.1:9876/ 就一目了然。

主视图。左 sidebar 是工作区分组,中间是卡片,右上一只走来走去的像素小猫念诗。

关键的一句话

这是一个给人看的仪表盘,不是一个让 AI 来查询的接口。 AI 只负责把 session 聚类成卡片;每张卡片的"读"还是你自己来。 我试过让 AI 直接回答"我最近在做什么",结果它要么瞎编要么照抄 mindmap.json 内容,都不如你自己扫一眼卡片来得快。

它会自动做什么

  • 每次 Claude Code 结束一轮对话(Stop hook),触发后台跑 Layer 1 总结当前 session
  • 触发 Layer 2 把所有 hot session 跨会话归类(coalesced,15 分钟冷却,Haiku ~$0.17/次)
  • 每 2h 生成 20 条 tips(每条带可点击的 source URL,wisdom 类用真实诗句,curiosity 类引维基/etymonline 等可核验来源 — 防 AI 瞎编)
  • 每周五中午自动出周报
  • 无人看 dashboard 时全部 idle,没有后台轮询烧钱

单卡放大。三态任务(待办 / 已完成 / 已取消)、卡点提示、artifact 直接点开 CR。

它不会自动做什么

  • 不会替你写代码、不会替你做技术决定
  • 不会"知道你最近在 Claude 整体花了多少钱"(stray --cost 只统计本插件自己的 AI 调用开销,不是你 Claude Code 的整体用量)
  • 不会"暂停 Claude"。它能暂停的只是本插件自己的后台 pipeline (stray --pause "demo 备战"),Claude Code 本体照常工作
  • 不收集任何会话内容到外部 — 唯一的网络出口是调用 Anthropic API 做分类,数据本身留在你机器

那只猫

本来只是想把右上角的 tips 文字做得不那么严肃,加了一个 emoji 头像。 后来觉得 emoji 太轻,换成了 CC0 的像素猫 sprite(感谢 OpenGameArt 的 nicolae-berbece), 6 帧 walk cycle 一直走。你可以拖它到屏幕任意位置,位置会存在 localStorage。 点它就换一条 tip。

Tips 不让 AI 瞎编

诗句类(wisdom)偏写景闲适,不要励志说教。采菊东篱下,悠然见南山 比 莫等闲白了少年头 让人放松。每条都必须配 source URL(我让它 指向 Wikipedia / Wikiquote 这类可核验的页面),AI 找不到可信链接就直接 drop 那条,不许编。

小知识类(curiosity)同理,必须配 Etymonline / Wikipedia / MDN / 官方文档之类的源。Markdown 是 2004 年 John Gruber 创立的 — 链接到 daringfireball.net。 Git commit hash 是 SHA-1 — 链接到 Wikipedia git 页面。

装它

一行装(macOS / Linux):

curl -fsSL https://raw.githubusercontent.com/Icesource/claude-stray/main/bin/quick-install.sh | bash

脚本会做依赖检查(python3 ≥ 3.9、git、登录过的 claude CLI), clone 到 ~/.claude-stray/(像 ~/.fzf / ~/.nvm 一样,工具自己的家目录),装完 Claude Code 的 Stop / SessionStart hook,顺手装 SKILL(让你后续问 "帮我打开仪表盘" / "刷新一下" / "这插件本月花了多少钱" 时主 agent 知道怎么调)。

不想 pipe shell 的话:

git clone https://github.com/Icesource/claude-stray.git ~/.claude-stray
cd ~/.claude-stray
bash bin/install.sh

装完打开 Claude Code 跑一次 /stray-refresh 出初始数据, 然后 stray --serve 看仪表盘。

开销

大致这样(Haiku 4.5,你自己的 Claude Code 订阅就行,不需要单独 API key):

  • Layer 1 总结:每个新 session 约 $0.04,Stop hook 触发
  • Layer 2 分类:约 $0.17/次,合并触发(15 分钟冷却 + dirty tracking),活跃使用约 5 次/天
  • Tips:每 2 小时 ~$0.08,只在 dashboard 跑着的时候
  • 周报:周五一次,$0.10-$0.50,看那周内容多少

都有 --max-budget-usd 硬上限,跑出预算会失败而不是悄悄烧钱。stray --cost month 看本插件这个月花了多少。

不打算做的事

有些功能我刻意没做:

  • session 全文搜索 — 那是 grep 的活
  • 聊天 UI / 让你直接对话 dashboard — 你已经有 Claude Code 了
  • 团队共享 / 多人协作 — 这是个人工具,session 数据本身就是私密的
  • "自动写代码替你解决卡点" — 太大胆,留给未来 P 系列做 agent 实验时再说
